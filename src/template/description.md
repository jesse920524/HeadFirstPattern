###模板方法模式（Template Pattern）

定义:
模版方法定义了一个算法的步骤,并允许子类为一个或多个步骤提供实现.

###模版方法的应用

数组的排序算法. sort()方法内部控制了算法的实现,
但sort()要求子类正确的实现compareTo()方法.


###最后 模版方法模式解决了什么问题?

不好的实现:
 
多个子类各自控制算法.
多个子类间存在重复的代码.
改变算法需要修改每个子类的代码.
算法的实现分散在许多类中. 

模版方法改进: 

基类控制算法,并保护这个算法.
由于基类控制算法, 可将代码复用最大化.
改变算法只需要修改基类的代码.
基类专注算法本身,由子类提供算法的完整实现.
